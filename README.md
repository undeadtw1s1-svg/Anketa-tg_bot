# Универсальный Telegram-бот для сбора заявок

Простой, быстрый и понятный бот на aiogram 3 для автоматизации приема заказов. Подходит для любого бизнеса или фриланса.

### Как это работает:
1. Пользователь запускает бота и выбирает нужную услугу через готовые кнопки (не нужно писать текст руками).
2. Бот просит кратко описать детали или пожелания.
3. Бот собирает все ответы в одну четкую анкету и выдает готовый результат.

### Как запустить бота?
Нужно установить требования, вставить свой ключ и запустить main.py
