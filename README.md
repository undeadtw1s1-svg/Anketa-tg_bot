# Асинхронный Telegram-бот для заказа услуг

Удобный и быстрый Telegram-бот, написанный на Python, который автоматизирует процесс обработки клиентских заявок.

## 🚀 Функционал
* **Интерактивное меню:** Быстрый выбор категорий или услуг с помощью встроенных кнопок.
* **Сбор информации:** Бот в диалоговом режиме запрашивает у пользователя детали и пожелания к заказу.
* **Формирование заявки:** Мгновенно собирает все данные в структурированный вид.

## 🛠 Технологии
* Python 3
* Aiogram

## Как запустить бота?
Чтобы запустить бота вам нужно токен в Botfather, вставить его в 6 строку в main.py, установить требования в requirements.txt и запустить main.py
